鼎捷应用研发代码生成器
设计方案：过滤器链 、mybaits代码生成器配置思想、模板式代码生成
     一、重写抽象过滤器 DWAbstractGeneratorFilter 用于各核心过滤器调度。
     实现不同功能过滤器：
             1、配置信息检查过滤器；
             2、API规格解析过滤器 ； 
             3、代码规格目录生成过滤器；
             4、代码生成过滤器；
             5、注解生成过滤器；
             6、其他；
     不同种类过滤器不同节点处理相应逻辑。通过过滤器链进行管理与顺序调度。
     选择过滤器链方案原因：程序运行过程中，需要在不同节点处理相应逻辑工作，工时可自定义配置部分过滤器的屏蔽与启用，
     过滤器间数据可以链式流转，保证全局配置、API信息数据在执行过程中分享使用。
